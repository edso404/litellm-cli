# Changelog

## v0.1.3 (2026-07-02)

### 新增功能
- **团队排行增强**：新增「用量排行」子Tab，展示团队成员的 Token 和请求数排名
- **时间范围选择**：支持 1-5 快捷键切换时间范围（最近7天/30天/3个月/半年/今年）
- **排序切换**：支持 o 键切换排序方式（Token / 请求数）
- **并发分页**：团队活动数据自动并发获取所有分页，响应更快速

### UI 优化
- 时间范围选择器可视化显示
- 显示具体日期范围（如：2026-01-01 ~ 2026-07-02）
- Sub-tab header 常驻显示

### Bug 修复
- 修复时间范围切换不生效的问题
- 修复 API key 映射逻辑（从 metadata 获取 key_alias）
- 修复分页数据不完整的问题
